## Getting Started

### Prerequisites
- Flutter SDK (3.1.0+)
- Dart SDK

### Installation
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Project Structure
- `lib/core/`: Core configurations (theme, routing, localization)
- `lib/features/`: Feature-specific modules
- `lib/shared/`: Shared components and utilities
